# Google Login PoC (Frontend)

## Prerequisites

- Node.js (v18+ recommended)
- `.env` file with:
```

VITE_GOOGLE_CLIENT_ID=
VITE_BACKEND_URL=

````

## Quick start

```bash
git clone <frontend-repo-url>
cd <frontend-folder>
npm create vite@latest . --template react
# or if repo already has setup:
npm install

# start development server
npm run dev
````
